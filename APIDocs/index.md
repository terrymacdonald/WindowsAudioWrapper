# WindowsAudioWrapper API Docs

Welcome to the generated API reference for **WindowsAudioWrapper**, a C# facade over the various Windows Audio APIs. Use this site to explore the public classes, helpers, and DTOs that make up the wrapper.

The goal of WindowsAudioWrapper project is to provide a lightweight, simpler way to access the many different Windows Audio APIs, and to store those in objects serialisable in JSON. That same JSON can be given to the library, and the library will then configure the Windows Audio Settings to make them reflect those passed settings.

## How to use

- Start at the API landing page: [WindowsAudioWrapper API Reference](/api/WindowsAudioWrapper.html).

## Where to Learn More

- **Project overview & usage**: See `README.md` at the repository root or the repo at https://github.com/terrymacdonald/WindowsAudioWrapper for quick-start examples and patterns.
- **WindowsAudioWrapper Samples**: Runnable samples demonstrating display, desktop, GPU, and event-listener flows are under `Sample_Code/`.
